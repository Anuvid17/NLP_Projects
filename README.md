## NLP Project 1

This project performs sentiment analysis on a customer reviews dataset using Natural Language Processing (NLP). The review text is cleaned and preprocessed before being analyzed with the VADER Sentiment Analyzer to classify sentiments as Positive, Negative, or Neutral. The project also extracts frequently occurring words, identifies common complaint-related terms, and generates visualizations such as sentiment distribution charts and a word cloud. The insights obtained from the analysis help in understanding customer opinions, measuring satisfaction levels, and identifying areas for improvement in products or services.

## NLP Project 2

This practical demonstrates the application of Natural Language Processing (NLP) techniques to analyze customer sentiments from restaurant reviews. The review text is preprocessed through text cleaning and stopword removal, and the VADER (Valence Aware Dictionary and sEntiment Reasoner) sentiment analyzer is used to classify each review as Positive, Negative, or Neutral. The results are further visualized using a sentiment distribution bar chart and a Word Cloud, providing insights into customer opinions and the most frequently used words in the reviews.

## NLP Project 3

Developed a sentiment analysis system to classify movie reviews into Positive, Neutral, and Negative categories using the IMDb Top 1000 dataset. The project involves data preprocessing, sentiment labeling based on IMDb ratings, audience opinion analysis, and visualization of sentiment distributions through charts and word clouds. It helps understand viewer preferences and overall movie reception.
